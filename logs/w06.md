# Learning Logs- W6

## CSS Centering & Layout
- Learned why `<img>` doesn’t always center:
  - Needs a width
  - Is inline by default → fix with `display: block`
- Ways to center a div:
  - Flexbox (easiest)
  - Grid
  - Margin auto (horizontal only)
  - Absolute positioning

## CSS Concepts
- Some styles override others (e.g., `font-family`)
- Important to understand selector priority

## CSS Layout Evolution
- Tables → Floats → Flexbox → Grid  
- Flexbox = 1D layout  
- Grid = 2D layout  
- Modern websites use both together

## Flexbox Basics
- Apply `display: flex` to parent container
- Children:
  - Align in a row by default
  - Adjust size automatically
- Useful properties:
  - `space-between` for spacing
  - Flexbox is the easiest way to center elements
