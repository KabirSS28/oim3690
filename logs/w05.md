Cascading in CSS, defines that the latest code will be used for the styling purposes 

## CSS Syntax

selector {
  property: value;
  property: value;
}
Example:
h1 {
  color: blue;
  font-size: 24px;
  text-align: center;
}

Basic Selector: 

Selector	 -        Targets	            -        Example
- Element	 -        All elements of type	-       p { }
- Class	     -        Elements with class	-       .intro { }
- ID	     -        Element with ID       -       #header { }
- Universal	 -        All elements	        -        * { }
