## Forms Basics
- Forms are used to collect user input through fields (input boxes)
- Common uses:
  - Sign up / Login
  - Search bars
  - Contact forms
  - Checkout pages

## The `<form>` Element
- Basic structure:
  ```html
  <form action="/submit" method="POST">

  </form>
