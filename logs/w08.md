# Week 8 Learning logs

## JavaScript Basics
- `console.log()` → used to print and test values in the console
- `addEventListener()` → waits for user actions (like clicks)
- Functions (e.g., `enlargeImage()`) → run code when triggered

## Symbols in JavaScript
- `{ }` → defines function scope
- `( )` → used for function inputs/parameters
- `[ ]` → stores multiple values (arrays)

## Variables
- `const` → value cannot change
- `let` → value can be updated

## DOM & Events
- `DOMContentLoaded` → runs code when page loads
- Used `querySelector` to select elements
- Example logic:
  - Image enlarges on click
  - If size > 200 → reset to 30


## CSS Frameworks
**Why use them:**
- Pre-built components (buttons, cards, forms)
- Consistent design
- Responsive by default
- Faster development

**Types:**
- Component-based
- Utility-first
